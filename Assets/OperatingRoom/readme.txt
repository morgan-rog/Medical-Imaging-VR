Operating Room - Stylized Props
The OperatingRoom folder contains all the models, prefabs, scenes, scripts, and select Standard Assets components (FirstPersonCharacter and CrossPlatformInput). 
FBXs, materials, and textures are located in the models folder.
Prebuilt prefabs for the models are included in the prefabs folder, which you can drag and drop into your scene.
An example scene is included in the scenes folder demonstrating how the props can be used.
Post Processing should be instlaled from the Package Manager to get depth of field and ambient occlusion post-process volumes.
